Welcome to Day 3 of JavaScript! This lesson covers how to make programs make smart choices, map execution directions using Control Flow frameworks, and resolve complex workflows using conditional structures.

## 📝 What You Will Learn
* How to compare values using JavaScript conditional expressions
* How to link multiple conditions cleanly using logical grouping syntax
* How to visualize application paths using programmatic control logic
* How to resolve true/false conditions safely with structural code execution loops

## 🛠️ Classroom Tasks

1. **Logical Operators Reference:** Explore how JavaScript manages complex comparisons using standard operator patterns:
   * `&&` (AND): Returns true only if every conditional block statement evaluates to true.
   * `||` (OR): Returns true if at least one block statement evaluates to true.
   * `!` (NOT): Inverts a boolean result cleanly.
2. **Comparison Operators Reference:** Examine how parameters match against relative scales:
   * Basic bounds checking: `>`, `<`, `>=`, `<=`
   * Identity checking: `===` (Strict equality matching for value and type)
3. **Control Flow Concepts:** Trace your application's logic steps. Learn why conditional test statements break code out of uniform top-to-bottom lines into distinct true/false forks.
4. **If / Else Code Structures:** Implement algorithmic branching syntax using conditional statement evaluation chains.
5. **Switch Statement Conversions (Optional):** Optimize lengthy conditional structures into scalable, highly readable item index filters.
6. **Dynamic String Alerting:** Reinforce template literal interpolation skills by packaging dynamic calculations into custom pop-up user interface alerts.
